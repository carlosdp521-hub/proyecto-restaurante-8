# Restaurante Sabores de Chile

Sitio web responsivo desarrollado para la actividad de Programación Web I.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- JavaScript

## Requisitos cumplidos

- Menú de navegación flexible.
- Uso de contenedor y elementos flexibles.
- Alineación y ordenamiento con Flexbox.
- Diseño adaptable para escritorio, tableta y smartphone.
- Media queries para pantallas menores a 768 px y 480 px.
- Galería de imágenes en la sección "Plato de la Semana".
- Formulario de reserva.
- Código organizado en carpetas.

## Estructura del proyecto

restaurante-sabores-chile-flexbox/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── README.md

## Instrucciones de uso

1. Abrir la carpeta del proyecto en Visual Studio Code.
2. Abrir el archivo `index.html`.
3. Ejecutar con Live Server o abrir directamente en el navegador.
4. Usar la herramienta de inspección del navegador para probar vista móvil y tableta.
